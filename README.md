# 🧬 NOVA BIOTECH | Elite Research Chile

**Plataforma Premium de Biohacking y Suministro de Péptidos para Investigación.**

Este proyecto es una web de una sola página (Single Page Application) diseñada con una estética minimalista tipo Apple, orientada a laboratorios e investigadores independientes en Chile.

## ✨ Características
- **Diseño Disruptivo:** Estética Cyber-Biotech blanca con cortes diagonales y scroll storytelling.
- **Optimizado para Chile:** Precios en CLP y cumplimiento de normativas locales de investigación.
- **Rastreo de Lote (COA):** Sistema simulado de verificación de certificados de análisis.
- **Responsivo:** Adaptado perfectamente para dispositivos móviles y escritorio.
- **Blindaje Legal:** Descargos de responsabilidad integrados para cumplimiento normativo.

## 🚀 Instalación y Despliegue
1. Sube el archivo `index.html` a tu repositorio de GitHub.
2. Ve a **Settings** > **Pages**.
3. En la sección "Build and deployment", selecciona la rama `main` y la carpeta `/root`.
4. Haz clic en guardar y tu web estará online en segundos.
